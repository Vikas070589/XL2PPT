How to Use:

  • Data: Use the 'Summaries' sheet; replace blanks with "NA" or "Not Available."
  • Inputs: Upload the Excel file and select a template type.
  • Output: Specify the path for the output folder.
  • Indexing: Set the start and end rows for processing.
  • Conversion: Optionally, choose to convert the output to PDF.
  • Click 'Submit' to run the tool.
